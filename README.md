# dc501@nmsu.edu-web-project

# Project 2 - HTML Foundation Page

# Description
This project is a simple wiki-style webpage about the book *Hyperion* by Dan Simmons. The site includes two linked pages that describe the story, main themes, and the different pilgrims. It is designed using only HTML to demonstrate basic webpage structure.

# What I Learned
While creating this project, I learned how to properly structure an HTML document using elements like headings, paragraphs, lists, tables, and images. I also learned how to link multiple pages together using anchor tags and how to organize content in a clear and readable way.

# Challenges
One challenge I faced was making sure all the links between pages worked correctly, especially with file paths. I also had to be careful to follow the requirement of using only HTML without any CSS. I love using CSS. Its time consuming but so awesome to see a cool looking web page.
